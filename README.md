# obstacle_avoider
	A ROS package for obstacle avoidance 

### To use the package:
	1) Open terminal
	2) cd ~/catkin_ws/src
	3) git clone https://github.com/hariharan20/obstacle_avoider
	4) cd ~/catkin_ws
	5) catkin_make
	6) roscore
### Now open a new terminal and type:
	rosrun obstacle_avoider obst.py
